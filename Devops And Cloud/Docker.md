### Docker Terminology
- Docker Image
  - Image = File System Snapshot + default run command

### Docker Commands
- check docker installation

```
docker version
```

- docker information
```
docker info
```

- run hello-world image
```
docker run hello-world
```

- list all docker images
```
docker images
```
